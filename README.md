# Site AZA Engenharia e Construção

Site estático preparado para publicação no GitHub Pages.

## Estrutura

- `index.html`: pagina inicial.
- `css/styles.css`: estilos do site.
- `assets/logos`: logos em SVG e PNG de referencia.
- `assets/images`: fotos usadas no site.
- `brand`: guia de identidade visual, tokens e referencias.

## Identidade Visual

O guia principal está em `brand/README.md`. A paleta foi organizada com vermelho como cor principal da marca e azul aço como detalhe complementar para dar um visual mais moderno e técnico.

## Publicacao no GitHub Pages

Ao subir o repositorio, configure o GitHub Pages para publicar a partir da branch principal e da raiz do projeto.
